# Support Conda installatoin 

## we recommend create a new conda environment for metaFun
conda create -c bioconda -c conda-forge -n metafun metafun
## If you have mamba, you can install metaFun with mamba
mamba create -c bioconda -c conda-forge -n metafun metafun
